# 🌳 Unreal Engine 5.5 to s&box Exporter (Полная Инструкция)

Этот гигантский и сложный Python-скрипт (`export_to_sbox.py`) автоматически портирует любые выделенные объекты (Static Meshes, Skeletal Meshes) из движка Unreal Engine 5 прямиком в ваш проект для s&box / Source 2, создавая 100% готовую к игре графику по одному клику.

---

## 🔥 Что скрипт делает "под капотом"?

1. **Зеркалирование папок**: Если ваше дерево лежало в папке `/Game/Nature/Trees/Pine_01/`, скрипт физически создаст точно такую же красивую иерархию папок внутри вашего проекта `s&box/Assets/...`. Никакой свалки из сотен картинок!
2. **Экспорт геометрии**: Тихий и безошибочный экспорт `.fbx` геометрии со всеми сглаживаниями и тангенсами.
3. **Генерация .vmdl (ModelDoc)**: Автоматическая сборка модели для Source 2, создание физической коллизии и привязка материалов.
4. **Умный парсинг материалов (PBR)**:
   - Автоматически вычисляет UV Scale из графа Unreal (на лету читая параметры `Texture Coord X` и `Y`).
   - Идеально "режет" тяжелые `_rma` / `_rsa` текстуры на 3 отдельных черно-белых маски (АО, Roughness, Metallic) и **сразу стирает за собой мусорные оригиналы**, освобождая десятки мегабайт.
5. **Магия Шейдеров (`complex.vfx`)**:
   - Автоматически включает **Alpha Test** прозрачность и двусторонний рендер для всех листьев.
   - Для непрозрачной коры автоматически включает 3D-рельеф (**Parallax Occlusion Mapping**) с глубиной `g_flHeightMapScale 0.030`.
   - Автоматически включает масштаб и самозатенение для микро-нормалей (что полностью сломало бы прозрачность листьев, но скрипт "видит" прозрачность и отключает эти галочки!).

---

## 🛠️ Подготовка (Один раз)

1. Откройте Unreal Engine. Зайдите в **Edit -> Plugins**.
2. В поиске вбейте `Python` и убедитесь, что плагин **Python Editor Script Plugin** включен.
3. В том же окне плагинов найдите **Editor Scripting Utilities** и убедитесь, что он тоже включен. Перезапустите движок, если потребовалось.
4. Проверьте, что ваш файл сохранен по пути: `H:\test\unreal_to_sandbox\export_to_sbox.py`.

## 🖥️ Вариант 1: Использование без кнопки (Через консоль)

Этот метод подходит, если вы хотите сделать быстрый разовый экспорт или если не хотите создавать кнопки.

1. В нижнем окне Unreal Engine (Content Browser) выделите мышкой одно или несколько деревьев (`Static Mesh`), которые хотите экспортировать.
2. В самом низу окна Unreal Editor слева найдите строку ввода консоли `Cmd`. Нажмите на надпись `Cmd` и переключите её режим на **Python**.
3. Скопируйте следующий код, вставьте его в эту консоль и нажмите **Enter**:

```python
import sys
if r"h:\test\unreal_to_sandbox" not in sys.path:
    sys.path.append(r"h:\test\unreal_to_sandbox")

import export_to_sbox
import importlib
importlib.reload(export_to_sbox)

# Путь до папки Assets в вашем проекте s&box! Замените, если он поменяется.
export_to_sbox.export_selected_to_sbox(r"H:\sbox\my_project_2\Assets")
```

👉 Как только вы нажмете Enter, скрипт на секунду "задумается" и в логе полетят строки успешного экспорта.

## 🚀 Вариант 2: Использование с кнопкой в UI (Выбор Профессионала)

Это лучший и самый удобный метод. Мы навсегда "зашьем" этот скрипт в интерфейс Unreal Engine. Вы сможете экспортировать объекты за 1 правый клик мыши.

1. В окне **Content Browser** (Браузер контента) нажмите правой кнопкой мыши по любой пустой серой зоне.
2. Выберите **Утилиты редактора** (Editor Utilities) -> **Блюпринт утилиты редактора** (Editor Utility Blueprint).
3. В появившемся окне выбора базового класса выберите **AssetActionUtility**. 
4. Появится новый синий файл блюпринта. Назовите его `BP_SboxExporter` и откройте двойным щелчком.
5. В открывшемся окне слева в разделе **Функции** (Functions) нажмите кнопку **+** (Плюс), чтобы создать новую функцию.
6. Назовите эту функцию **`Export To Sbox`**. (Именно так будет называться наша кнопка при клике мышкой!).
7. Справа в меню **Детали** (Details) найдите чекбокс **"Вызывать в редакторе"** (Call In Editor) и **ПОСТАВЬТЕ** галочку. Это самый важный шаг.
8. Возвращаемся в центр графа. От начального красного блока `Export To Sbox` вытяните мышкой белую нитку вправо и отпустите.
9. В строке поиска вбейте строго по-английски `Execute Python Command` и выберите эту зеленую ноду.
10. Кликните на спавнившуюся зеленую ноду. Справа в разделе Details найдите огромное текстовое поле **Command** (Команда) и вставьте туда следующий блок кода:

*(Код точно такой же, как и в варианте 1)*
```python
import sys
if r"h:\test\unreal_to_sandbox" not in sys.path:
    sys.path.append(r"h:\test\unreal_to_sandbox")

import export_to_sbox
import importlib
importlib.reload(export_to_sbox)

export_to_sbox.export_selected_to_sbox(r"H:\sbox\my_project_2\Assets")
```

11. Нажмите галочку **Compile** (Компилировать) слева вверху и кнопку **Save** (Сохранить). Восклицательные знаки должны исчезнуть. Блюпринт можно навсегда закрыть.

### 🎉 Как этим пользоваться:
1. Выделите любое количество моделей деревьев в окне Unreal Engine.
2. Нажмите по любому из них **Правой Кнопкой Мыши**.
3. В длинном списке контекстного меню найдите раздел **Действия скрипта с объектами** (Scripted Asset Actions).
4. Выберите **`Export To Sbox`**.

Всё! Скрипт мгновенно выполнит весь многоступенчатый PBR-экспорт на ваш жесткий диск! Гениальная автоматизация завершена.
